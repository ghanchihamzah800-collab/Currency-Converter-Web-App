# Currency Converter Web App

A simple web application that converts currency values in real time.

Live demo: https://ghanchihamzah800-collab.github.io/Currency-Converter-Web-App/

## What It Does
This app allows users to convert an amount from one currency to another using current exchange rates and displays corresponding country flags.

## Features
- Real-time currency conversion
- Automatic flag updates based on selected currencies
- Responsive design
- Input validation

## Technology Used
- HTML
- CSS
- JavaScript
- External exchange rate API
